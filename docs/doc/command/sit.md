::: tip 关于FSit
该功能可以直接使用  
你可以选择在客户端安装[FSit](https://modrinth.com/mod/fsit) Mod获得更好的体验  
:::

### **直接使用**
  
**右键** 半砖或楼梯————坐下，`Shift` 键下来  
  
**右键** 玩家————叠高高，`Shift` 键下来 或 让上面的人下来  

### **指令使用**
  
`/sit` ————在任意地方坐下，`Shift` 或 `/sit` 起来  
  
`/crawl` ————在任意地方爬行，再次 `/crawl` 起来
